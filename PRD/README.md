# 1-day 모델 알바 매칭 시스템 PRD 패키지

이 디렉토리는 1일 모델 파티/행사 알바 매칭 시스템을 위한 기획 및 제품 요구사항(PRD) 문서들을 포함하고 있습니다.
기능 구현, 데이터 모델링, 단계별 로드맵, 기술 스펙이 구체화되어 있으며, 개발을 시작하기 전 기준점 역할을 합니다.

## 파일 구성

1. [01_PRD.md](./01_PRD.md) - 제품의 목적, 타겟 유저, 핵심 기능 및 해결하려는 문제 정의
2. [02_DATA_MODEL.md](./02_DATA_MODEL.md) - 주요 데이터 구조 및 엔티티 관계
3. [03_PHASES.md](./03_PHASES.md) - 개발 단계별 로드맵 (MVP ~ 향후 확장)
4. [04_PROJECT_SPEC.md](./04_PROJECT_SPEC.md) - 기술 스택, UI/UX 원칙 및 구현 제약 사항

## 실행 가이드 (개발 시 참고)
* 이 문서는 모바일 웹 환경을 타겟으로 작성되었습니다.
* 토스(Toss) 앱 빌더 수준의 직관적이고 깔끔한 UI를 목표로 합니다.
* 당장의 결제 모듈보다는 매칭(구인 등록 -> 실시간 사진 인증 및 지원 -> 승인) 코어 루프 구현에 집중해야 합니다.

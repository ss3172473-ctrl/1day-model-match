# Product Requirements Document: 1-Day 모델 알바 매칭 시스템

## Summary
파티 주최자(고용주), MD(에이전시), 그리고 개인 모델을 연결하는 1일 알바 매칭 시스템입니다. 고용주가 1일 행사(파티 등) 모델 구인 글을 올리면, 모델 또는 MD가 프로필과 함께 '실시간 카메라 인증 사진'을 첨부하여 지원하고, 고용주가 이를 승인하는 직관적인 매칭을 제공합니다.

## Background & Objective
- **문제점:** 기존 파티/행사 모델 구인은 음지에서 이루어지거나, MD를 통한 카카오톡 연락 등 파편화되어 있어 신뢰도와 프로세스 추적이 어렵습니다. 또한 지원자의 사진이 현재 모습과 다를 수 있는(보정 등) 리스크가 큽니다.
- **목표:** 직관적인 모바일 웹 환경을 통해 구인-지원-승인의 매칭 절차를 시스템화합니다. 특히 '실시간 카메라 촬영 필수' 기능을 통해 지원자의 신뢰도를 높이고, MD와 개인 모델의 역할을 시스템적으로 명확히 분리하여 관리를 용이하게 합니다.

## Target Audience
1. **파티 주최자 (고용주)**
   - 행사에 필요한 모델을 특정 시간/장소/조건으로 구인합니다.
   - 지원한 모델들의 프로필과 실시간 사진을 보고 승인 여부를 결정합니다.
2. **개인 모델**
   - MD 없이 직접 플랫폼에 가입하여 자신의 프로필(키, 몸무게, 나이 필수)을 등록하고 공고에 지원합니다.
   - 지원 시 반드시 스마트폰 기본 카메라로 사진을 찍어 제출해야 합니다.
3. **MD (에이전시)**
   - 다수의 소속 모델 프로필을 자신의 계정에서 등록하고 관리합니다.
   - 소속 모델을 대신해 조건에 맞는 구인 공고에 지원하며, 주최자와의 소통을 전담합니다.

## Solution & Key Features (MVP Scope)
- **모바일 웹 최적화 인터페이스:** 토스 앱빌더 스타일의 깔끔하고 직관적인 UI/UX 제공.
- **역할 기반 회원 시스템:** 회원가입 시 주최자 / 모델 / MD 유형 선택.
- **구인 공고 작성 (주최자):** 
  - 필수 항목: 행사 일시, 상세 장소, 일당, 인원수, 추가 요구 사항(복장, 컨셉 등 자유기재).
- **프로필 관리 (모델/MD):**
  - 필수 항목: 키, 몸무게, 나이, 기본 프로필 사진.
  - 선택 항목: 경력, 인스타그램 링크, 추가 소개 등.
- **실시간 카메라 인증 지원 (핵심):**
  - 지원하기 버튼을 누르면 기기의 기본 카메라가 활성화되어 현재 얼굴/모습을 촬영해야 지원이 완료됩니다.
- **지원자 리스트 및 승인 (주최자):**
  - 공고별로 지원한 사람들의 프로필과 실시간 사진을 카드 형태로 확인하고 '승인' 버튼으로 채용 확정.

## Non-goals (MVP 제외 대상)
- **플랫폼 내 결제 및 정산 시스템:** 주최자가 모델이나 MD에게 지급하는 알바비 및 수수료 정산은 시스템 외부에서 처리 (추후 논의 후 도입).
- 실시간 채팅 기능: (초기에는 연락처 교환 방식이나 단순 매칭 상태 메시지로 대체할 수 있으며, 필요 시 Phase 2 도입).

## Risks and Constraints
- **실시간 카메라 제어:** 모바일 웹 환경 브라우저에서 HTML5 `<input type="file" capture="user" accept="image/*">`를 사용하여 기기 카메라를 직접 호출하도록 구현해야 합니다. 브라우저 및 OS별 지원 여부 확인 필요.
- **MD와 모델의 권한 분리:** MD가 생성한 모델 프로필은 MD만 제어 가능해야 하며, 일반 모델 유저와 데이터적으로 분리 또는 연결되는 구조가 명확해야 합니다.
- **연락 없는 매칭:** 주최자가 승인하면 별도의 연락 없이 사전에 공지된 장소와 시간으로 모델이 찾아가는 방식이므로, 노쇼(No-Show) 발생 시 대처 방안이 추후 필요할 수 있습니다.
